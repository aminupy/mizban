# Support

## Getting help

- Usage and installation: `README.md`
- Release process: `docs/RELEASE.md`
- Throughput/performance validation: `docs/THROUGHPUT_TEST_PLAN.md`

## Community and issue support

- Bug reports: https://github.com/aminupy/mizban/issues/new
- Feature requests: https://github.com/aminupy/mizban/issues/new

When opening an issue, include:

- OS and architecture
- Mizban version
- reproduction steps
- expected and actual behavior
- logs/screenshots if relevant

## Security issues

For vulnerabilities, follow `SECURITY.md` and use private advisories instead of public issues.
